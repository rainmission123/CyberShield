# CyberShield

CyberShield is an Android security app built with Kotlin. It helps users scan suspicious links, detect potentially malicious URLs, and show warning alerts before opening unsafe websites.

## Features

* URL scanning
* VirusTotal integration
* Scan history
* Warning screen for suspicious links
* Accessibility-based link detection
* Alert sound and vibration
* Clean Android UI

## Tech Stack

* Kotlin
* Android Studio
* XML Layouts
* VirusTotal API
* SharedPreferences / Local Storage
* Accessibility Service

## Purpose

This project was created as a cybersecurity-focused Android portfolio app. It demonstrates mobile security awareness, URL threat detection, and Android app development skills.
